# 这不对啊

群魔中，五人小组在犯下残忍罪行后，才终于发现，这不对啊。  
这不对啊，这一切都不对啊，这荒诞世界。

在被行刑的那五分钟，远处教堂的金顶光辉。与世界告别！  
可是在大赦天下之后，又浪费了许多分钟

梅诗金公爵，面对这痛苦的悲剧，兄弟杀害爱人，如何拯救这一切？

《伊凡·伊里奇之死》对死亡的刻画，简直让人困惑：托尔斯泰明明没死过，怎么能写出死前的心理？

伊万在与魔鬼的对话，自我拉扯。

默尔索面临审判，这究竟是对谁、对什么的审判？要因为我没有在母亲葬礼上哭泣而审判我吗？神父究竟为什么执着于救赎我？没有救赎，没有告解。  
这刺眼灼人的阿尔及利亚海滩上的阳光，就是我开枪的原因。
